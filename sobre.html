Sobre Nós

Este site é a concretização de um projeto acadêmico e comunitário, desenvolvido por Matheus Prado e João Vitor, estudantes do CENTRO UNIVERSITÁRIO INTERNACIONAL - UNINTER
Nosso objetivo ao criar este espaço foi duplo: não apenas aplicar os conhecimentos que adquirimos em desenvolvimento web durante nossa graduação,
mas também oferecer uma ferramenta ou recurso valioso para a comunidade.
Acreditamos que a tecnologia tem o poder de transformar e conectar pessoas, e este projeto é a nossa contribuição para isso

Cada funcionalidade, foi pensada para ser útil, refletindo nosso compromisso com o impacto social positivo. Este é fruto de estudo e dedicação para a comunidade.
Esperamos que você aproveite este espaço que preparamos com tanto carinho. Se tiver alguma sugestão ou feedback, adoraríamos saber!
